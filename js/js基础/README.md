# js 基础
## i++
在 js 中的 i++ 跟 go 中的 i++ 不同，go 中的 i++ 表示 i = i + 1，然后返回的是 i，而 js 中的 i++ 表示 ii= i，i = i + 1，然后返回的是 ii，跟 go 相似的是 js 中的++i 表示 i = i + 1，然后返回的是 i
